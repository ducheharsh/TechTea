# 🚀 TechTea Quick Start Guide

Get TechTea up and running in 5 minutes!

## Step 1: Installation (30 seconds)

```bash
# Clone the repository
git clone https://github.com/yourusername/techtea.git
cd techtea

# Install dependencies
npm install
```

## Step 2: Run Development Server (10 seconds)

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Step 3: First Time Setup (2 minutes)

### Create Your Profile
1. Enter a username on the landing page
2. Click "Enter TechTea"

### Create Your First Room
1. Click "Create Room" button
2. Fill in:
   - Room name (e.g., "Web Dev Warriors")
   - Description (e.g., "Learning React together")
   - Choose Public or Private
3. Click "Create Room"

### Post Your First Update
1. In the text area, write your update
2. Use Markdown for formatting:
   ```markdown
   # Today's Learning
   - Learned React Hooks
   - Built a todo app
   
   ```javascript
   const [state, setState] = useState()
   ```
   ```
3. Click "Post Update"

## Step 4: Add Integrations (Optional, 2 minutes)

### GitHub Stats
1. Get token: [GitHub Settings](https://github.com/settings/tokens)
2. Click "Integrations" in room
3. Paste token, click "Save"

### Wakatime Stats
1. Get key: [Wakatime Settings](https://wakatime.com/settings/api-key)
2. Add in Integrations panel
3. Click "Save"

### Coding Platforms
1. Enter usernames for:
   - LeetCode
   - HackerRank
   - CodeChef
2. Click "Connect"

## Step 5: Explore Features (3 minutes)

### Check Leaderboard
- Click "Leaderboard" button
- See rankings and AI insights
- Switch between Today/Week/All Time

### Share Your Progress
1. Click "Share" on any update
2. Choose template (Minimal/Detailed/Stats)
3. Download image or share to social media

### Try Social Features
- React to updates with emojis
- View room stats
- Explore members list

## 🐛 Troubleshooting

### Can't see updates?
- Refresh the page
- Check room membership
- Clear browser cache

### PWA not installing?
- Use HTTPS (production)
- Check browser compatibility
- Try Chrome/Edge for best experience

### Integration not working?
- Verify API keys are correct
- Check token permissions
- See [SETUP.md](./SETUP.md) for details
